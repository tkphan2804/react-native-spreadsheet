# react-native-spreadsheet

This is spreadsheetView for react-native 

## Installation

```sh
npm install react-native-spreadsheet
```

## Usage

```js
import Spreadsheet from "react-native-spreadsheet";

// ...

const result = await Spreadsheet.multiply(3, 7);
```
